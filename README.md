# 📁 File Explorer

A lightweight file explorer built in C# that provides a simple and efficient way to browse, manage, and interact with the file system. The project focuses on usability, clean navigation, and basic file operations similar to Windows File Explorer.

---

## ✨ Features

- 🌳 Directory tree navigation
- 📂 File and folder browsing
- 📄 File details display (size, type, modified date)
- ✏️ Basic file operations:
  - Rename
  - Delete
  - Move / Copy
- 🔍 Fast directory switching
- 🖱️ Intuitive UI with event-driven interaction


---

## ⚙️ Requirements

- .NET Framework / .NET 8.0.0

---

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/OrMichal/File_Explorer.git
```

2.
```bash
cd File_Explorer/Sunrise_Terminal
```

3. Run the project
```bash
dotnet run
```
